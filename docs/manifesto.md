# Manifeste Lumeni

Nous croyons que les commerces de proximité créent chaque jour des centaines de petits moments qui rendent une ville plus humaine.

Un café offert.

Un prénom retenu.

Un dessert d'anniversaire.

Un sourire.

Ces moments ne devraient jamais être oubliés.

Lumeni existe pour aider les commerçants à les faire vivre.

---

Les meilleurs logiciels gèrent des données.

Les meilleurs produits créent des émotions.

---

Le commerçant est le héros.

Jamais le logiciel.

---

Chaque écran doit raconter une histoire.

Chaque statistique doit être utile.

Chaque clic doit faire gagner du temps.

Chaque fonctionnalité doit rapprocher le commerçant de ses clients.
